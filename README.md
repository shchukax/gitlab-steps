# Gitlab Steps Plugin
Jenkins (workflow) plugin that exposes limited Gitlab API via Jenkins workflow methods.

Currently supported methods:
* Create branch
* Get tags
* Create tag
* Create pull request
* Merge pull request
* Create/update file
